# Changelog

All notable changes to this project will be documented in this file. This project adheres to [Semantic Versioning](https://semver.org/).

## [1.1.0] - 2025-11-17
### Added
- UAEP protocol integration (core, production, staging mode files)
- Agent profiles (Laravel, Security) reorganized under `codex/prompts/`
- New CLI loader: `cx-uaep` to activate UAEP protocol
- UAEP documentation, usage examples, and project structure overhaul
- README.md fully updated for UAEP usage and guidance

### Changed
- Project structure streamlined under `codex/` and `bin/`

### Deprecated
- Old `templates/` directory (replaced by `codex/prompts/`)

## [1.0.0] - 2025-05-09
### Added
- Initial release: cx-commit, cx-review, cx-fix, cx-security CLI helpers
- Core templates for agent profiles
- Project README and setup instructions
